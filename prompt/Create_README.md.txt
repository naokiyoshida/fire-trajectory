# 重要：出力形式の制約（必ず守ること）
1. README.md の全内容を、最初から最後まで一つの「text」コードブロック（```text ... ```）で囲んで出力してください。
2. 内部で「```」を使用すると、表示システムがコピー範囲を誤認して途切れるため、内部ではバッククォートを使用しないでください。
3. アーキテクチャ図は Mermaid 構文ではなく、アスキーアート（テキスト図解）で作成してください。
4. 数式は LaTeX 形式ではなく、プレーンなテキスト（$r = (1+n)/(1+i)-1$ など）で記載してください。
5. これにより、ユーザーが右上のコピーボタン一つで、1文字も漏らさず内容を抽出できるようにします。
